# 🐱⚡ Bella's Pokémon Spelling Adventure

A custom spelling game built for **Bella** — Unit 4, Week 5, Soft C & G Words.

🎮 **[Play the Game](https://adamnichols59.github.io/pokemonspelling31625/)**

> **Best experience:** Open in Safari on iPhone → tap Share → **Add to Home Screen** for full-screen app mode!

---

## 🎮 How to Play

### Screen 1 — Spell the Word
- Listen to the word, sentence, and word again (tap 🔊 to replay)
- Type the word directly into the letter tiles
- Tap ✅ CHECK when ready
- 💡 Hint shows the first and last letter if you're stuck

### Screen 2 — Catch the Creature!
- A kawaii creature appears on screen
- A colored ring shrinks on the creature — just like Pokémon Go
- **Swipe the ball upward** to throw it
- 🟢 **Green ring** = Great Throw! (+3 bonus pts)
- 🟡 **Yellow ring** = Nice! (+1 bonus pt)
- 🔴 **Red ring** = Normal catch
- ⚫ **Full size** = Miss! Try again

### Screen 3 — Map Moves Forward
- Your trainer walks to the next location on the world map
- 5 locations to unlock across 20 words

## 🔊 Audio

- Each word is spoken aloud: **word → sentence → word**
- Background chiptune music
- Sound effects: encounter sting, ball throw, catch fanfare, victory music
- Tap 🔈 in the top bar to mute/unmute
- Voice powered by [Puter.js](https://puter.com) — free, no account needed for players

---

## 🛠️ Tech

- Single `index.html` file — no build tools, no installs
- All 20 creatures drawn in hand-coded SVG
- Sound effects and music generated with Web Audio API (no audio files)
- Pokémon Go–style ring throw mechanic built with `requestAnimationFrame`
- Keyboard push-up uses `visualViewport` API (works in Safari/Chrome on iPhone)
